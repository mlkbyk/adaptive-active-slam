01 // System Architecture

Modüler Yazılım Katmanları Modular Software Layers

9 Nodes Detected
Node_01

circle_generator.cpp

Robotun etrafında (360°) dinamik çözünürlüklü aday hedef noktaları üretir. Generates dynamic resolution goal candidates around the robot's perimeter.

Node_02

gain_evaluator_node.cpp

Hedeflerin keşif potansiyelini ve engellere olan güvenli mesafesini puanlar. Scores exploration potential and safety clearance for each candidate goal.

Node_03 (Core)

slam_health_monitor.cpp

AMCL kovaryansını ve TF stabilitesini izleyerek sistemin güven skorunu hesaplar. Calculates system confidence by monitoring AMCL covariance and TF stability.

Node_04

decision_manager_node.cpp

Otonomi modları (Explore, Recover, Pause) arasındaki geçiş mantığını yönetir. Governs the state transition logic between Explore, Recover, and Pause modes.

Node_05

motion_executor_node.cpp

Seçilen hedefleri move_base'e iletir ve acil durum protokollerini denetler. Transmits selected goals to move_base and enforces emergency protocols.

Node_06

candidate_visualizer_node.cpp

RViz marker verilerini gerçek zamanlı olarak görselleştirmekten sorumludur. Responsible for real-time visualization of markers and scores in RViz.

Node_07

acs_control_panel_node.cpp

Otonomi döngüsüne manuel müdahale ve anlık durum izleme arayüzü sağlar. Provides a manual intervention and live status monitoring interface.

Node_08

metrics_logger_node.cpp

Performans analizleri için sistem verilerini CSV formatında kayıt altına alır. Logs system performance metrics in CSV format for post-analysis.

Node_09

circle_scoring.cpp

Aday puanlama süreçlerinde kullanılan çekirdek matematik kütüphanesidir. The core mathematical library utilized for candidate scoring processes.
